This repository contains all coursework, assignments, and projects completed as part of ISYE 6740: Computational Data Analytics, a core course in the Master of Science in Analytics program at Georgia Tech.

The course focuses on computational methods for analyzing large and complex datasets. Topics include data wrangling, algorithmic approaches for data analysis, optimization techniques, predictive modeling, and high-performance computing for analytics, with an emphasis on building scalable and efficient solutions to real-world data problems.
